ビルドは`build`ディレクトリの中でやってください

# How to build

```sh
pwd # /home/[USERNAME]/.../data_structures/build ← buildにいることを確認
```

```sh
cmake .. # Makefileの生成. `CMakeLists.txt`を更新したときはこれを実行
```

ビルドするよ

```sh
cmake --build .
```

実行するよ

```sh
./data_structures
```

# 実装した方が良いデータ構造

- 配列
- 連結リスト
- スタック
- キュー
- ハッシュテーブル
- ツリー
- グラフ
- ヒープ
- トライ木
- セグメントツリー
- フェニックストリー
- バランス木 (AVL木, 赤黒木など)
- その他興味のあるデータ構造
